Audio with non-zero Start Time

This player is initialized to start automatically, at 2:20 into the track (just prior to the guitar solo). This can be used to start the media from search results or an interactive transcript. To play the media automatically at the target start time, the media element must also include the autoplay attribute. Even then, some browsers may block this functionality.
