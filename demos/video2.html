Video with default closed captions

This page was designed for testing the assignment of a default caption track. The following is the order of precedence:

  1. The first caption track that has a default attribute
  2. The caption track that matches the language of the player
  3. The first caption track

In the current example, the language of the player is English, but there's a default attribute on the French track, so French is the default language for the captions and transcript.

If the user has turned captions off on the current website, they will remain off, but the default caption language will be honored in the transcript.

Another purpose of this page is to test for an early bug in some browsers that resulted in browsers' native controls being displayed, even if no controls attribute was present, when one of the tracks had a default attribute. For additional details about this bug see Able Player Issue 5.
